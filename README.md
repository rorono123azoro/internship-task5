# Internship Task 5: Database Integration

Integration of a SQL database (SQLite) into an existing Express-based REST API to learn data persistence.

## Features
- SQL Integration using node-sqlite3 or similar
- Robust CRUD operations modifying actual database records
- Storing and retrieving User entities persistently

## How to run
1. Ensure Node.js is installed.
2. Run `npm install` to install dependencies.
3. Start the application using `node server.js`. The database file will be initialized automatically.
